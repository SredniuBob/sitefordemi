Установка Demi

Установите Demi на LozzitaiOS или PulseOS с помощью нескольких простых команд

В LozzitaiOS Demi уже предустановлен. Для обновления до последней версии выполните:

# Обновление списка пакетов
sudo demi update

# Обновление самого Demi до последней версии
sudo demi install --self-update

# Обновление всех пакетов системы
sudo demi upgrade

Примечание: В LozzitaiOS Demi полностью заменяет apt и является пакетным менеджером по умолчанию.

В PulseOS Demi также предустановлен. Для настройки и обновления выполните:

# Настройка репозиториев PulseOS
sudo demi configure-pulseos

# Обновление Demi и системы
sudo demi update && sudo demi upgrade

# Установка дополнительных инструментов разработчика
sudo demi install pulseos-dev-tools

Примечание: Demi в PulseOS включает расширенные функции для разработчиков и облачной интеграции.

Для других дистрибутивов на базе Ubuntu/Debian:

# Добавление репозитория Demi
curl -fsSL https://demi.pkg/ubuntu/key.gpg | sudo gpg --dearmor -o /usr/share/keyrings/demi-archive-keyring.gpg

# Добавление источника пакетов
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/demi-archive-keyring.gpg] https://demi.pkg/ubuntu $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/demi.list > /dev/null

# Установка Demi
sudo apt update
sudo apt install demi

Основные команды Demi

Команды, оптимизированные для LozzitaiOS и PulseOS

  • demi install <пакет> Установка пакета и его зависимостей (в LozzitaiOS/PulseOS автоматически выбирает оптимизированные версии)
  • demi remove <пакет> Удаление пакета с сохранением конфигурационных файлов
  • demi update Обновление списка доступных пакетов из репозиториев LozzitaiOS/PulseOS и Ubuntu
  • demi upgrade Обновление всех установленных пакетов до последних версий
  • demi search <запрос> Поиск пакетов в репозиториях LozzitaiOS/PulseOS
  • demi lozzita-update Обновление только основных пакетов LozzitaiOS (без обновления приложений)
  • demi pulseos-dev Установка полного набора инструментов разработчика PulseOS
  • demi list-lozzita Показать все установленные пакеты LozzitaiOS
  • demi list-pulseos Показать все установленные пакеты PulseOS
  • demi configure-pulseos Настройка Demi для работы с репозиториями PulseOS
  • demi rollback Откатить последнее обновление пакетов (создает точку восстановления)
  • demi snapshot create Создать снимок текущего состояния пакетов
  • demi snapshot restore <id> Восстановить состояние пакетов из снимка
  • demi clean Очистить кэш загруженных пакетов
  • demi stats Статистика использования пакетного менеджера
